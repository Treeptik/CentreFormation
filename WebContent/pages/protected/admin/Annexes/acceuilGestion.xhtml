Accueil Gestion:
  • Un stagiaire peut être créé, modifié.
  • Son mot de passe de passe est généré automatiquement, il est envoyé à l'e-mail enregistré.
  • Un Stagiaire est inscrit aux sessions via le Module des sessions.
  • Un Formateur peut être créé, modifié.
  • Un Formateur Fictif (nom: Evaluation , prenom: Generale) est le seul lié à la Formation "Evaluation Generale" (voir explication Module Formation).
  • Un Formateur est lié aux formations qu'il est susceptible d'animer via le Module Formation
  • Gestion : Une formation peut être créée, modifiée.
  • Formateur : Lier les formateurs susceptibles d'animer cette formation, lister, supprimer.
  • Evals : Ce sont les formations composant une session qui sont évaluées.
  • Pour l'instant le fonctionnement impose pour être optimal:
    • DANS CETTE ORDRE : A une formation classique est liée Evalution Formation, Evaluation Formateur et Commentaire Generale.
    • Une formation fictive "Evaluation Generale" est nécessaire au fonctionnement du système d'évaluation.
    • Cette formation est liée à l'évaluation "Evaluation Generale" (indépendante des formations).
    • Il faut, pour l'instant, lier cette formation en PREMIER à toutes sessions.
  • Stagiaire : Un Stagiaire est inscrit aux sessions via le Module des sessions.
  • Formation : Une Formation est liée à une ou plusieurs sessions.
  • Une Evaluation peut être créée, modifiée.
  • Question : Lier les questions, lister, retirer de l'évaluation.
  • Pour l'instant le fonctionnement impose:
    • Créer Evaluation Generale liée aux questions indépendantes des formations (accueil, matériel, salle).
    • Créer Evaluation Formation liée aux questions : rythme , sujet à approfondir, à supprimer etc...
    • Créer Evaluation Formateur liée aux questions : écoute, disponibilité capacité technique, pédagogique.
    • Créer une Evaluation Commentaire Generale lié uniquement à la question Commentaire Generale pour pouvoir laisser une zone de commentaire libre.
  • Une Question peut être créée, modifiée.
  • Une Question peut être liée à une évaluation via le Module Evaluation.
